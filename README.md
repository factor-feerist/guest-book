# Приложение "Гостевая книга"
Располагается по адресу https://d5d9v3s3qul2vtp0qn8p.yl4tuxdu.apigw.yandexcloud.net/

Показывает список гостей, предоставляет возможность внести запись о себе в список гостей, а также отображает актуальную версию бэкенда и фронтенда

## index.py
Основной код приложения

## requirements.txt
Зависимости для кода

## index.html
Статический фронт приложения, аналогичный лежащему в Object Storage на момент проверки

## for-serverless-guest-book.yml
Спецификация для API Gateway

## scripts.txt
Скрипты с инструкциями для автоматизации развёртывания приложения (PowerShell)

## guests.yql
Скрипт для создания схемы с гостями в YDB
